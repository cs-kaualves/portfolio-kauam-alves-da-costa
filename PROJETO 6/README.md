## 📌 Descrição

**OrcaFlow** é uma aplicação web de gestão financeira corporativa que centraliza o controle de orçamentos, despesas, aprovações e relatórios em um único painel. O sistema conta com um módulo de **AI Insights** em tempo real para análise preditiva de gastos e suporte à tomada de decisão financeira.

Desenvolvida inteiramente na plataforma **Bubble**, demonstra o poder do desenvolvimento no-code para construção de produtos SaaS complexos sem uma linha de código backend.

---

## 🎯 Funcionalidades

### 📊 Financial Command Center (Dashboard)
Visão consolidada em tempo real de toda a saúde financeira da organização:

| Métrica | Descrição |
|---------|-----------|
| **Active Budgets** | Total de orçamentos ativos (ex: 30 orçamentos em 12 departamentos) |
| **Total Allocated** | Envelope orçamentário do período fiscal (ex: FY 2026) |
| **Total Spent** | Percentual consumido do plano (ex: 65% do plano aprovado) |
| **Remaining Balance** | Saldo remanescente com indicador de headroom (ex: 35% de margem) |

### 🗂️ Módulos do Sistema

| Módulo | Função |
|--------|--------|
| **Dashboard** | Visão geral e KPIs financeiros em tempo real |
| **Budgets** | Criação e gerenciamento de orçamentos por período e departamento |
| **Expenses** | Registro e controle de despesas aprovadas |
| **Approvals** | Fluxo de aprovação de gastos (com contador de pendências) |
| **Reports** | Geração de relatórios financeiros |
| **Audit Log** | Histórico completo de ações para conformidade e rastreabilidade |
| **Settings** | Configurações da workspace |

### 🤖 AI Insights
- Análise automática de padrões de gastos
- Alertas proativos (ex: *"Travel spend is pacing 22% higher than Q4 2025"*)
- Recomendações de ação (ex: revisar aprovações pendentes)
- Módulo **Live** com investigação guiada de anomalias financeiras

## 🖥️ Interface

A aplicação possui uma interface dark moderna com:

- **Sidebar de navegação** com acesso rápido a todos os módulos
- **Cards de KPI** com indicadores visuais de status (*On track*, percentuais, badges)
- **Utilization by Category** — gráfico de barras de alocação vs. gasto real por categoria
- **Painel de AI Insight** destacado na sidebar do usuário
- **Seletor de período fiscal** e botão de criação rápida (*+ New Budget*)
- **Perfil do usuário** com cargo e acesso a configurações

## 🚀 Como Acessar

Por ser uma aplicação Bubble, não requer instalação local. O acesso é feito diretamente pelo navegador via URL da aplicação publicada.

> Para testar localmente durante o desenvolvimento, utilize o **Debug Mode** disponível na barra inferior do Bubble Editor (modos: Normal, Slow, Step-by-step).
